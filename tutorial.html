Instructions for BonkTheDog Project

Step 1: Set up the HTML structure

  • Create a new HTML document.
  • Define the document type (<!DOCTYPE html>).
  • Open the <html> tag and set the language attribute to "en".
  • Inside the <html> tag, open the <head> section.
  • Add meta tags for character set and viewport settings.
  • Link two external CSS files: "main.css" and "reset.css".
  • Set the title of the document to "BonkTheDog".

Step 2: Create the main container structure

  • Open the <body> tag.
  • Create a <div> element with a class of "container-window".
  • Inside the "container-window" div, create a <section> element with a class of "container".
  • Inside the "container" section, create three main divisions: "leftBox", "middleBox", and "rightBox".

Step 3: Populate the leftBox

  • Inside the "leftBox", create an <h1> element with a class of "clickCounter" and an empty <p> tag.
  • Create a <div> with a class of "parent" for grid layout.
  • Inside the "parent" div, create eight <div> elements with classes "side-dog", "side-dog-right", "side-dog-down", or "side-dog-up", each containing an <img> tag with the source pointing to "/assets/img/bonk.gif".
  • Add a <div> with a class of "main-dog" containing an <img> tag with the source pointing to "/assets/img/daggoe-mlg.jpg".

Step 4: Populate the middleBox

  • Inside the "middleBox", add a <p> tag with the text "2".
  • Create a <div> with a class of "row-1".
  • Inside "row-1", add three <img> tags with the source pointing to "/assets/img/daggoe-cry.jpg" and alternate text "dog".

Step 5: Populate the rightBox

  • Inside the "rightBox", create four <button> elements with IDs "dogBat", "middleRowButton1", "middleRowButton2", and "middleRowButton3", each labeled "Slappy dog 20 Credits".

Step 6: Add JavaScript functionality

  • Link an external JavaScript file "app.js" just before the closing </body> tag.
  • Implement JavaScript logic to handle button clicks and counter updates.
  • Add an event listener to the "dogBat" button to handle clicks.
  • Implement logic to check if enough credits are available and display appropriate messages.
  • Implement event listeners for clicking on the main dog image to increase the counter.
  • Use setInterval to update the counter value periodically.

Step 7: Style the layout using CSS

  • Define styles for the main sections ("section"), ensuring they are displayed as a flex container with appropriate spacing and alignment.
  • Style the container window with a fixed width and height, and a background color.
  • Define styles for the left, middle, and right boxes with specific widths, heights, and background colors.
  • Implement CSS grid layout for the "parent" container with three rows and three columns.
  • Define styles for hiding and showing images using the "imgHide" and "imgShow" classes.
  • Style individual dog images and rows as needed.

Step 8: Add specific styles for dog positions and rows

  • Define specific styles for each dog position (dog1 to dog8) to position them within the grid.
  • Define styles for the row-1 element to ensure proper display of images within the row.

Once these steps are followed, your HTML document will be structured, styled, and functional according to the provided code.
