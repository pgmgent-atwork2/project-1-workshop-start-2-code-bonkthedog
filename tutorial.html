Initialization Variables:

Two variables, imageIndex and counterValue, are initialized to keep track of the index of displayed images and the current counter value. 

let imageIndex = 0;
let counterValue = 200;

Window Load Event:

When the window has fully loaded, this event handler selects all image elements within each row and sets their source to /assets/img/daggoe.jpg. 

window.onload = function() {
    const rows = ['row1', 'row2', 'row3', 'row4', 'row5', 'row6'];
    rows.forEach((row, index) => {
        const images = document.querySelectorAll(`.${row} img`);
        images.forEach((img, imgIndex) => {
            img.src = `/assets/img/daggoe.jpg`; // replace with your image source
        });
    });
};

Main Dog Click Event:

This event listener increments the counterValue by one and updates the text content of an element with the class clickCounter to display the new value when the main dog image is clicked. 

document.querySelector(".main-dog img").addEventListener("click", function() {
    counterValue++;
    document.querySelector(".clickCounter").textContent = counterValue;
});

Counter Update Interval:

This interval function updates the counterValue every second by adding imageIndex to it, then updates the text content of the element with class clickCounter to reflect the new value. 

setInterval(function() {
    counterValue += imageIndex ;
    document.querySelector(".clickCounter").textContent = counterValue;
}, 1000);

Button Event Listeners:

These functions add event listeners to various buttons. They check if the counterValue is sufficient to purchase the corresponding item. If so, they reveal the hidden image and update the counterValue accordingly. 

function addEventListenerToButton(buttonId, requiredCredits, imgClass) {
    document.getElementById(buttonId).addEventListener("click", function() {
        if (counterValue >= requiredCredits) {
            const images = document.querySelectorAll(`.leftBox ${imgClass}`);
            const hiddenImages = Array.from(images).filter(img => img.classList.contains('imgHide'));
            if (hiddenImages.length > 0) {
                hiddenImages[0].classList.remove("imgHide");
                hiddenImages[0].classList.add("imgShow");
                imageIndex++;
                counterValue = Math.max(0, counterValue - requiredCredits); 
            }
        }
    });
}
function addEventListenerToButton2(buttonId, requiredCredits, imgClass) {
    document.getElementById(buttonId).addEventListener("click", function() {
        counterValue++; // increment counterValue each time button is clicked
        document.querySelector(".clickCounter").textContent = counterValue;

            const hiddenImages = Array.from(images).filter(img => img.classList.contains('imgHide'));

                hiddenImages[0].classList.remove("imgHide");
                hiddenImages[0].classList.add("imgShow");

}

Plus One Animation Interval:

This interval function creates a "+1" animation every second in image containers, then removes it after one second. 

setInterval(function() {
    const imageContainers = document.querySelectorAll('.image-container');
    imageContainers.forEach(container => {
        const plusOne = document.createElement('div');
        plusOne.textContent = '+1';
        plusOne.classList.add('plus-one');
        container.appendChild(plusOne);

        setTimeout(function() {
            container.removeChild(plusOne);
        }, 1000); // remove after 1 second
    });
}, 1000);
